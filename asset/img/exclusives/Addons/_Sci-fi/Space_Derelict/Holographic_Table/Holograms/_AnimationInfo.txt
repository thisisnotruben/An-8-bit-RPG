Notes:

These hologram animations can be overlaid on any of the tables available in this add-on.

You can play all the animation rows consecutively for a long and varied animation cycle.

However, each row is a separate animation that can be displayed separately.

List of animation types ordered by row top to bottom:

- Hologram appear
- Cycle 1
- Glitch 1
- Glitch 2
- Transition to cycle 2
- Cycle 2
- Transition to cycle 1
- Hologram disappear
