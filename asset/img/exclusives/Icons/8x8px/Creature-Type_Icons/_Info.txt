Each row contains icons to represent a creature type.

Each creature type has different icon options, so you can choose the one that best meets your needs.

Each row also has the respective icons in the following available variations:

- Only silhouette
- Coloured
- With outline

Using the visual guide included (Guide.png), you will be able to easily identify the icons that belong to each creature type.

The creature types included are:

- Humanoid
- Beast
- Monster
- Undead
- Construct
- Elemental
- Divine
- Demon
- Aberration
- Plant
- Giant
- Ooze
- Magical
